## üöÄ Quick Start

### Prerequisites

- Node.js >= 20.0.0
- npm, yarn, pnpm, or bun

### Installation

```bash
# Clone the repository
git clone https://github.com/srteerra/keno.git
cd keno

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.
